1.黏贴备份好的博客文件夹
2.安装 git nodejts hexo等插件
3.去管理员: Windows PowerShell界面执行:查看当前执行策略:Get-ExecutionPolicy
4.同界面下执行：设置新的执行策略，例如将执行策略设置为 RemoteSigned:Set-ExecutionPolicy RemoteSigned

然后应该就没啥问题了,再有毛病那就超纲了得去另请高明了hhh
